第一部分   商业数据挖掘基础（第1-3章）

CRISP-DM 数据挖掘标准流程

方法论流程框架

跨行业数据挖掘标准流程，业界最广泛使用的数据挖掘方法论，包含6个阶段的迭代循环。

  • 业务理解 — 明确商业目标与分析需求
  • 数据理解 — 数据收集、描述与质量评估
  • 数据准备 — 清洗、转换、特征工程
  • 建模 — 选择算法、训练与调参
  • 评估 — 模型效果验证与业务解释
  • 部署 — 模型上线与持续监控

维度建模与指标体系

数据架构业务口径

基于星型/雪花模型构建企业级数据架构，通过事实表与维度表分离，实现灵活的业务分析。

  • 事实表 — 存储可度量的业务事件数据
  • 维度表 — 描述业务实体的属性信息
  • 指标分层 — 原子指标 → 衍生指标 → 复合指标
  • 数据血缘 — 指标口径管理与一致性保障

特征工程与数据预处理

数据处理建模基础

数据质量决定模型上限，特征工程是数据挖掘中最耗时却最关键的环节。

  • 缺失值处理 — 删除/填充（均值、中位数、模型预测）
  • 异常值检测 — 3σ法则、IQR法、孤立森林
  • 变量转换 — 标准化、归一化、对数变换
  • 特征构造 — 交叉特征、多项式特征、业务衍生
第二部分   描述性分析与商业洞察（第4-5章）

可视化分析与经营驾驶舱

数据可视化BI

将数据转化为直观的图表与仪表盘，支撑数据叙事与管理决策。

  • 图表选择 — 比较用柱状图、趋势用折线图、占比用饼图
  • KPI监控 — 北极星指标 → 一级指标 → 二级指标
  • BI工具 — Power BI / Tableau / FineBI

RFM模型与客户价值分层

客户分析精准营销

基于R(最近消费)、F(消费频率)、M(消费金额)三个维度对客户进行精细化分层。

  • 高价值客户 — R低 F高 M高 → 重点维护
  • 潜力客户 — R低 F低 M高 → 激活消费频率
  • 流失预警 — R高 F低 M低 → 召回策略

关联规则与购物篮分析

无监督学习零售分析

发现商品之间的购买关联关系，驱动交叉销售与商品组合优化。

  • 支持度(Support) — P(A∩B)，规则出现的频率
  • 置信度(Confidence) — P(B|A)，规则的可靠性
  • 提升度(Lift) — P(B|A)/P(B)，规则的实用性
  • Apriori算法 — 频繁项集 → 强关联规则
第三部分   预测性分析与风险预警（第6-8章）

分类、回归与聚类方法总览

监督学习无监督学习

根据业务目标选择合适的建模方法，是数据挖掘的核心决策。

  • 分类 — 逻辑回归、决策树、随机森林、XGBoost
  • 回归 — 线性回归、岭回归、时间序列模型
  • 聚类 — K-Means、DBSCAN、层次聚类

时间序列分析与需求预测

预测模型供应链

基于历史数据预测未来趋势，为库存优化与补货决策提供数据支撑。

  • 趋势(Trend) — 长期上升/下降方向
  • 季节性(Seasonality) — 周期性波动模式
  • 残差(Residual) — 随机波动成分
  • 常用模型 — ARIMA、Prophet、LSTM

信用评分卡与金融风控

风控建模金融科技

构建评分卡模型量化评估信用风险，实现自动化审批决策。

  • WOE编码 — 证据权重转换，处理分类变量
  • IV值 — 信息量评估，衡量变量预测能力
  • 评分刻度 — Base Score + PDO转换
  • 异常检测 — 孤立森林、LOF算法、AutoEncoder
  • 模型解释 — SHAP值、LIME局部解释
第四部分   智能决策与综合应用（第9-12章）

文本挖掘与情感分析

NLP非结构化数据

从文本、评论、社交媒体等非结构化数据中提取商业洞察。

  • 文本预处理 — 分词、去停用词、词性标注
  • 特征提取 — TF-IDF、Word2Vec、BERT嵌入
  • 情感分析 — 正面/负面/中性情感分类
  • 主题建模 — LDA主题模型发现隐含话题
  • 社交网络分析 — KOL识别、信息传播路径

A/B测试与实验设计

因果推断运营优化

通过对照实验科学验证业务策略效果，驱动数据化运营决策。

  • 假设检验 — H0原假设 vs H1备择假设
  • P值 — 观测到当前结果的概率（< 0.05显著）
  • 样本量估算 — 效应量、显著性水平、统计功效
  • 分流策略 — 随机化、AA验证、辛普森悖论

生成式AI与智能决策

前沿技术AI赋能

生成式AI正在重塑数据分析范式——从手动编写代码到AI辅助自动分析。

  • AI辅助编程 — CodeBuddy/Copilot自动生成分析代码
  • 智能问数 — 自然语言提问 → AI自动生成图表与洞察
  • 自动化报告 — AI生成数据分析报告初稿
  • AI治理 — 模型透明性、公平性、问责机制

敏捷分析与智能决策闭环

决策科学智能化

从传统"分析→报告→决策"线性流程，升级为AI时代的智能决策闭环。

  • 数据采集 → AI自动分析 → 洞察生成 → 决策行动 → 效果反馈 → 持续迭代
  • 关键突破 — 将AI嵌入决策流程，缩短"数据→洞察→行动"周期
